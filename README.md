# Sohail Writes

A modern, SEO-optimized blog aggregator built with Next.js 16, featuring a bold Neo-Brutalism design system. Aggregates and showcases articles from multiple platforms (Medium, Hashnode, Dev.to) with powerful search and filtering capabilities.
